01. Core Business & Application Features

The application is an industry-grade B2B portal offering Embroidery Digitizing and Vector Conversion services to commercial printers, embroiders, and corporate clients.

📦

Order Lifecycle Management

End-to-end workflow from submitting a quote request or placing a direct order, through to design execution, review, correction logs, and production-ready deliveries.

  • Customer uploads source artwork
  • Admin reviews & assigns to specific vendor
  • Vendor delivers embroidery/vector versions
  • Customer files revisions or downloads deliverables
👥

Three-Tiered Access Control

Role-based dashboard systems custom-tailored for the three principal users of the ecosystem.

  • Customers: Manage designs, pay invoices, choose membership levels, and checkout carts.
  • Vendors: Receive job assignments, report stitches, request charges, and download design files.
  • Admins: Oversee order workflows, manage price rules, view sales graphs, and track receivables.
⚡

Dynamic Pricing Engine

Highly configurable pricing mechanism taking calculations from user levels, parameters, and time deadlines.

  • Tiers: Regular, Silver, Gold, Platinum plans.
  • Stitch Ranges: Tiered rates (e.g. 1-10k, 10-20k, etc.)
  • Vector Complexity: Simple vs. Complex multipliers.
  • Surcharges: Delivery speed variables (Rush / Standard).
📊

Unpaid Orders & Statement Reports

Comprehensive admin ledger utility for monitoring outstanding revenue and receivables.

  • Real-time dashboard summaries for overdue counts and unpaid order aggregates.
  • Color-coded warnings (Yellow for unpaid, Red bold highlighting for overdue).
  • Grace periods automated based on customer's Payment Terms account tier.
  • Integration with automated SMTP notification reminders.
☁️

AWS Integrations & Compression

Built-in storage optimizations and cloud integration utilities to support large file uploads smoothly.

  • Secure integration with AWS S3 using SDK APIs for artwork distribution.
  • Auto-compression of raw design attachments directly inside client browser.
  • Encrypted tokenized API endpoints safeguarding data flows.
  • Structured file names preserving order and revision logs.

03. Secure Payment Gateway Workflows

The portal supports secure credit-based and terms-based transaction methods, operating primarily via Authorize.Net APIs and PayPal.

Visual Workflow Flowchart Interactive Path

💡 System Architecture Flow Detail

When a customer processes items in their cart, the system checks whether they possess a custom corporate account status ("On Terms"). For standard retail orders, credit cards are charged immediately or authorized to protect delivery. For terms accounts, invoices accrue with grace periods calculated relative to order timestamps.

💳

Credit Card Flow (Authorize.Net Integration)

Direct card operations run on backend server endpoints (api/payment.js & api/paymentAuth.js) via secure API requests.

  • Authorization Capture (authCaptureTransaction): Runs for normal shop checkouts. Instantly charges the customer's card and marks the design order status as "Active".
  • Authorization Only (authOnlyTransaction): Specifically sets a temporary reserve hold limit up to $50.00 to verify account solvency before full design files are released.
  • Fallback Error Mailing: If payment API throws exceptions (incorrect cvv, insufficient funds), backend triggers api/paymentErrorEmail, alerts the user, and sends diagnostic details to admins.
📝

Payment Terms Accounts ("On Terms")

Business accounts registered under pre-negotiated invoice cycles bypass credit card gates during basic design checkouts.

  • Grace Period Management: Uses date-fns library utility to check registration grace levels (e.g. Net 15, Net 30, Net 45 days).
  • Ledger & Overdue Badging: Automatically marks orders as "Overdue" using prominent red visual rows in report dashboards the moment current date values slip past due limits.
  • Balance Statements: Generates dynamic monthly PDF summaries using jspdf and `xlsx` plugins, keeping clients informed of outstanding invoices.
