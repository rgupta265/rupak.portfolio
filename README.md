# Deploy to Vercel (Free)

**Fastest way (no git needed):**
1. Go to https://vercel.com/new
2. Sign up free (GitHub/Google/email)
3. Drag & drop this folder (or just `index.html`) into the upload box
4. Click Deploy — done. Live URL in ~30 sec.

**Via GitHub (recommended for future edits):**
1. Create a new GitHub repo, upload `index.html`
2. On vercel.com/new, "Import Git Repository" → select repo
3. Framework preset: "Other" → Deploy

**Custom domain (optional, free subdomain):**
After deploy, go to Project Settings → Domains → edit to `rupakgupta.vercel.app`
